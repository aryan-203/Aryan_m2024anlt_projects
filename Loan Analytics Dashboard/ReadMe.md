## Introduction
This project presents a visually-rich Power BI dashboard that provides a summary of an organization's loan portfolio. It highlights performance metrics like sanctioned amount, disbursed amount, requested loans, ROI, loan cancellations, and customer segmentation.

## Data Origin
The dashboard uses operational bank loan data, capturing details of thousands of loans such as branch, customer profile, amounts sanctioned/disbursed, property type, ROI, and cancellation status.

## Tools Used
- Power BI (for dashboard visualization)

## Dashboard Features

- **Key Metrics Display:** Instant overview panels show the number of loans approved, cumulative sanctioned amount, total disbursed amount, requested amount, and average ROI.
- **Trends Visualization:** Line charts display growth in amounts sanctioned/disbursed over time.
- **Segmentation:** Bar charts and tables break down loans by customer profile, branch, property type, and lead source.
- **ROI and Distribution:** Pie/block charts illustrate ROI segments and average annual distribution.
- **Cancellation Analysis:** Donut chart summarizes the proportion of loans canceled versus successfully disbursed.
- **Record Table:** Scrollable data table lists every loan entry with all relevant details for granular inspection.

## Insights Generated

- Analysis of YOY sanctioned and disbursed amounts reveals business growth and operational trends.
- Customer and property segmentation highlight where the portfolio is strongest and where risks may lie.
- Average loan ROI helps in benchmarking against internal or industry targets.
- The dashboard helps spot patterns in cancellation rates for process improvement.

## Usage

1. Open the dashboard file in Power BI Desktop.
2. Use filters/slicers to focus analysis by branch, year, customer profile, etc.
3. Explore charts for summary insights and drill down into the detailed loan records table for specifics.

## Conclusion

This dashboard enables rapid, data-driven portfolio analysis for financial institutions, supporting continuous improvement and strategic planning. Power BI's interactivity makes it easy to explore key business questions with just a few clicks.
